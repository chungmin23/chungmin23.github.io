# 입력,출력

### 입력

~~~java
//import문 추가
import java.util.*;

//Scanner 객체의 생성 
Scanner scanner = new Scanner(System.in);

//Scanner 객체를 사용 
int num = scanner.nextInt();

String input = Scanner.nextLine();
// 문자열을 숫자로 변환
int num = Integer.parseInt(input);

~~~



### 출력

~~~
//줄바꿈 하지 않음
System.out.print();

//출력하고 줄 바꾸기
System.out.println();

//형식대로 출력하기
System.out.printf("%s%n", "Hello JAVA");
~~~
